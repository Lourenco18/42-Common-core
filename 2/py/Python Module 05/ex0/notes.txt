class DataProcessor(ABC)->
    ABC vem de Abstract Base Class (Classe Base Abstrata). 
    É um recurso do Python para definir classes que não devem ser 
    instanciadas diretamente, mas servem como modelo para outras classes.

Method Overriding -> 
    Está implementado na superclasse.
    Subclasses não são obrigadas a implementá-lo.
    Se a subclasse não sobrescrever, ela herda o comportamento da superclasse.
    Se a subclasse quiser sobrescrever, pode criar um método com o mesmo nome
